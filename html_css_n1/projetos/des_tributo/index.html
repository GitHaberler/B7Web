Chilly Willy

Meu apelido de casa vem justamente deste pequeno pinguim, pois sempre que meus pais me chamavam para ver o desenho eu vinha correndo.

Assim com o tempo foi ficando olha o Chilly para cá, Chilly para lá.

Claro que quando comecei a estrever meu apelido acabei mudando um pouco a grafia, mas o som é praticamente o mesmo, ficando Tile Willy no lugar de ser escrito Chilly Willy.

by Cristian Horst Haberler - 2020
